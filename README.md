# DesafioTarget

Minha solução para o desafio é estruturada da seguinte forma. A solução para cada um dos 3 problemas é implementada num projeto diferente. A solução para o problema N está no projeto DesafioTargetN. Para os dois primeiros problemas fiz uma API REST, em que a solução está em um dos endpoints do Controller. Para o problema 3, fiz um Console app.
